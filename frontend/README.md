# Frontend (React + TypeScript + Vite)

Этот каталог содержит клиентскую часть приложения pro100gym, разработанную с использованием React, TypeScript и Vite. Для стилизации используется Tailwind CSS.


## Установка

Чтобы запустить проект локально, выполните следующие шаги:

1.  **Перейдите в каталог `frontend`:**

    ```bash
    cd frontend
    ```

2.  **Установите зависимости:**

    ```bash
    npm install
    ```

## Запуск проекта

Для запуска dev-сервера выполните:

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173/`.

## Сборка для продакшена

Для сборки приложения для продакшена выполните:

```bash
npm run build
```

Файлы сборки будут сгенерированы в каталоге `dist/`.

## Линтинг

Для проверки кода на наличие ошибок линтинга:

```bash
npm run lint
```
